## What it is
Our program acts like a Letterboxd clone. You can input a movie and get a list of top 10 movies that are similar to it. You also have the option to rate the movie and write a review, which the program stores. If this were an actual, fully fledged website, users would be able to browse reviews. What makes it so fascinating is the way it groups movies together. It bases similarity on factors such as genre, director, and cast, to find similarities. 

## How to run it
The program is simple to run. You simply have to make sure you are on the mini_project tab, and on the terminal. In the top right corner there will be a play button. You just need to click that and wait for the program to load and begin! Use your keyboard to answer the questions, and press the ENTER key to mvoe on.

## How it works
The biggest part of the code is the dataset full of the movies and their information, which was found online. The code's job was to sort through the lists and categories to see which movies have similar/the same keywords. There were a lot of new pieces that had to be imported in, that couldn't be skipped as they were essential to the project working, that caused some difficulty. However, with research and learning the basics of installation, trial and error was the only thing needed to fix the code's errors.
